    Acme-Win32-PEPM a unique Perl+XS packager

    By bulk 88 from NY.pm
    Date: Saturday, 11 April 2015 11:00
    Duration: 55 minutes
    Target audience: Intermediate/Advanced
    Language: English
    Tags: binary image_loader memory os_design packaging shared_libraries windows

    A dive into mixing perl code with binary code, and ensuring both are parsable as each other. In this talk you will discover how to melt Perl code and Windows's DLL (shared library) formats together into 1 file, and the many challenges I faced in designing the most unique Perl+machine code packager on CPAN. This talk touches on Window OS's image loader, memory manager, extending ExtUtils::MakeMaker and the Visual C and Mingw GCC's linkers.
